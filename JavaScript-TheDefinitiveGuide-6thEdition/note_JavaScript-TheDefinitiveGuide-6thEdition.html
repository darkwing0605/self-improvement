数字

按照JavaScript中的数字格式，能够表示的整数范围是从-253~253，包含边界值。

整型直接量

十进制、十六进制（0x或0X开头）、八进制（0开头），对于八进某些JavaScript支持有些不支持，最后必要用，ES6严格模式下，八进制是明令禁止的。

浮点型直接量

[digits][.digits][(E|e)[(+|-)]degits]
例如
6.02e23
1.47E-32
